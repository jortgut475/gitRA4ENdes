  1. Que es un sistema de control de versiones?

    un software que ayuda a gestionar y controlar los cambios realizados en el código fuente u otros archivos de un proyecto a lo largo del tiempo

  2. Cual es la diferencia entre git y github?

    Git es el sistema de control de versiones en sí mismo, mientras que GitHub es una plataforma basada en la web que utiliza Git para alojar repositorios de código y proporciona herramientas adicionales para la colaboración y gestión de proyectos

  3. Cómo creo una rama en git?

    git branch [nombre-rama]

  4. Cómo clono un repositorio?

    git clone

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    git pull

  6. Cómo puedo gener un tag?

    git tag[nombre-del-tag]

  7. Cómo hago una mezcla de ramas?

    git merge [nombre-rama]

  8. Cómo puedo moverme a un commit especifico?

    git commit -m "Mensaje del commit"

  9. Cómo puedo borrar una rama?

    git branch -d

  10. Cómo puedo borrar un tag?

    git tag -d

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?
